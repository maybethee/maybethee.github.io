PROJECTS

Rock Paper Scissors

Simple best of five rock paper scissors game against a computer. Very early project with almost no styling or polish.

Calculator

A working calculator! Fully equipped with cutting-edge spaghetti code that *probably* covers *most* edge cases, as well as a sassy retort when you try to divide by 0!

Drawing Canvas

A canvas you can "draw" on with your mouse cursor. Another early project, but with a decently cohesive design, if rudimentary. Features changable pixel drawing size, and a rainbow option!

User Dashboard

Model webpage for a CSS styling project. No working functionality, and not particularly responsive, but much more visually polished than the other projects (so far).

Library

An exercise in using Javascript constructors to add Book info cards to a "Library". Not much emphasis on the visuals, but for the first time, some movement!
